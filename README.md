# Expensive Tracker

## 1. Description du Projet

**Objectif** : Développer une application web complète permettant aux utilisateurs de gérer et d'analyser leurs dépenses personnelles de manière organisée et visuelle.

L'application permettra aux utilisateurs de catégoriser leurs dépenses par thème (rooms) et de générer des statistiques détaillées sur leurs habitudes de consommation.
